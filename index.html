markdown-cv

a simple template to write your CV in a readable markdown file and use CSS to publish/print it.

View on GitHub Download .zip Download .tar.gz

Tired of using LaTeX to make your CV look pretty? Can't remember how to set margins in your TeX file? Refusing to compromise and use Word?

Meet markdown-cv, a simple template to list all your accomplishments in a readable Markdown file and use CSS to style it.

Instructions

To start, simply fork the repository and type your skills, jobs and education in the index.md file.

Then let your local installation of jekyll or GitHub Pages do the work of transforming your plain text CV into a beautiful looking HTML page. Type jekyll serve and you'll be able to see your CV on your local host.

Go to the page with your favourite browser and to get a PDF, just press print. Print and web CSS media queries should take care of the styling.

The included CSS renders CV in a style inspired by kjhealy's vita template, but any styling is possible. Contributions and forks are welcome!
